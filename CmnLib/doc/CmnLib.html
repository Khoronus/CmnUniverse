CmnLib

This is a library with common functionalities between projects.

cmnlibcore

Core classes and functions for memory and error management.

cmnlibworld

Main library interface (world) with headers to include in other projects.

container

Different type of container for fast access to sparse data (nearest key) or tree (i.e. Splay tree).

control

Classes for the log and control inforamtion.

string

Classes for operation with strings (iterator, conversion, tokenizer).

system

Classes for the system query (environment, console, time).

ts

Classes to test the system (ts).

version

Current version of the library.

sample

Collection of samples for the library.
