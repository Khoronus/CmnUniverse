CmnMath

This is a library with common mathematical functionalities between projects.

algebralinear

Collection of classes and functions to perform linear algebra operation such as: matrix operation, 2D/3D operations, vectors, shrink/saturation/round/square root.

cmnmathcore

Library with the definition of the atomic functions, constants, types, and thread safe structures.

cmnmathworld

Library with the world header to include all the projects at once.

coordinatesystem

Set of classes and functions to change the coordinate system (i.e. cartesian to polar).
